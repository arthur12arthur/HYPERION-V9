Architecture des 9 Agents

Agent A : Orchestrateur

Gestionnaire central du flux de travail et des priorités.

Agent B : Extraction

Cascade intelligente : HTML → PDF → Gemini Vision.

Agent C : Enrichissement

Collecte des données externes multi-sources.

Agent D : Analytique

Moteur de calcul, Monte Carlo et consensus Borda.

Agent E : Fusion

Méta-fusion des prédictions internes et externes.

Agent F : Risque

Gestionnaire HADES et calcul EV/Kelly.

Agent G : Reporting

Génération des rapports narratifs et techniques.

Agent H : Évaluation

Analyse des résultats réels et scoring J/30.

Agent I : Monitoring

Surveillance de la santé du pipeline et alertes.
