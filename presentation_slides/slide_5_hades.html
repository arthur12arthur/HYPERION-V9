Système HADES (Gestion des Risques)

Hazard Detection

Le Hazard Detection & Evaluation System est le gardien du capital. Il analyse les anomalies statistiques et les divergences entre les agents pour prévenir les pertes.

Phase 1 : Mode Test (J1-J30)

Analyse passive et logging des anomalies. Aucune restriction sur les prédictions pour calibrer les seuils d'alerte.

Phase 2 : Mode Production

Blocage automatique des recommandations de mise en cas de risque élevé ou de divergence forte du consensus.
